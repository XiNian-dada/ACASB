ACASB API Test Console

这是一个独立的接口测试页面。默认指向生产地址 http://server.hairuosky.cn:1111， 支持 JWT Token、数据集接口、大屏接口、互动接口、图片分析上传，以及任意自定义请求。

默认 Base URL: http://server.hairuosky.cn:1111
鉴权: Bearer Token
适合: 联调 / 回归 / 线上排查

环境配置

默认已经填好云端 Java 地址。部署后你只要粘贴启动日志里的 Bearer Token 就能开始测。

使用提示

  • 部署后如果只暴露 Java 端口，这个页面依然能正常测，因为 Python 服务只在内网被 Java 调用。
  • 浏览器端联调时，请优先测 `/api/dataset/*`、`/api/dashboard/*`、`/api/experience/*` 和 `/api/analyze`。
  • 所有 `/api/**` 和 `/data/**` 请求默认都要带 Bearer Token。
  • 如果跨域访问失败，请确认线上 Java 服务已经正常启动且没有被反向代理拦掉 `Authorization` 头。
  • 这个页面是纯静态文件，可以本地双击打开，也可以丢到 Nginx 静态目录里使用。
